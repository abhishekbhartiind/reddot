# reddot
Project simil Reddit
Backend: Express, GraphQL, Apollo Server, TypeORM, Redis, PostgreSQL.
Frontend: NextJS, React, Typescript, GraphQL, URQL, ChakraUI, Formik.

Features so far:

<ul>
  <li>Register</li>
  <li>Login</li>
  <li>Logout</li>
  <li>Auth / Cookies</li>
  <li>CreatePost</li>
  <li>UpdatePost</li>
  <li>DeletePost</li>
  <li>VotePost (+1 or -1)</li>
  <li>Cache</li>
  <li>Pagination</li>
</ul>
